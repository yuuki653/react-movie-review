# Movie Review App

## 主な機能（予定）
- TMDB APIによる映画情報取得
- 映画検索・詳細表示
- レビュー投稿・お気に入り登録
- Firebase認証・保存

## 技術スタック
- React
- React Router
- TMDB API
- Firebase
